1. 数位分离：输入一个整数（不定长），要求从个位开始依次倒序输出每一个位

```C++
int m;
cin >> m;
while (1){
    cout << m % 10; // 输出个位
    m = m / 10;
}
```



2. **ASCII字符可以被看作一个整数，可以直接运算、比较**

```C++
char ch = 'A';
if (ch > 10){
    cout<<"11";
}
ch = ch + 1;
if (ch % 2 == 0) cout << "";
else cout << "";
```
