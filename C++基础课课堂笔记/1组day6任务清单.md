【1】从键盘输入一个任意的三位正整数n，分别求出其个位、十位和百位上的数字。依次输出个位、十位、百位上的数字。

> 提示：输入数字n
>
> 个位：n % 10
>
> 十位：n / 10 % 10
>
> 百位：n / 10 / 10 % 10
>
> 定义三个变量，分别存储个位、十位、百位，

```C++
样例输入：123
样例输出：321
```

```C++
#include <iostream>
using namespace std;
int main(){
    int n, ge, shi, bai;
    cin >> n;
    ge = n % 10;
    shi = n / 10 % 10;
    bai = n / 100;
    cout << ge << shi << bai;
}
```
