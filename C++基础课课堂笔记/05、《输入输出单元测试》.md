
2. 输出第二个整数（1.1.02）

```C++

cin >> a >> b >> c;
cout << b;
```



3. 计算(a+b)*c的值（1.3.02）

5. 带余除法（1.3.04）


6. 输出保留3位小数的单精度浮点数（1.1.04）

```C++

```
